## Technical Issues

### Notifications Not Working

**Subject:** Re: Notifications not working - Troubleshooting steps

**English:**
```
Hi [Name],

I'm sorry to hear you're having trouble with notifications. Let's get this sorted out!

Please try these steps:

1. Check notification permissions:
   - Go to Settings > MicroBreaks
   - Make sure Notifications are enabled
   - Enable "Allow Notifications"

2. Check Do Not Disturb:
   - Ensure DND mode is not active
   - Or add MicroBreaks to allowed apps

3. Battery optimization (Android):
   - Go to Settings > Apps > MicroBreaks
   - Select "Battery"
   - Choose "Unrestricted" or "Not optimized"

4. Restart the app:
   - Close MicroBreaks completely
   - Reopen and check if notifications work

5. Reinstall (last resort):
   - Uninstall MicroBreaks
   - Reinstall from App Store/Google Play
   - Note: This will reset your data

If the issue persists after trying these steps, please let me know:
- Your device model
- OS version
- When the issue started

Best regards,
[Agent Name]
MicroBreaks Support
```

### App Crashing

**Subject:** Re: App crashing - Let's fix this

**English:**
```
Hi [Name],

I'm sorry you're experiencing crashes. Let's work together to resolve this.

Please try these steps:

1. Update MicroBreaks:
   - Check App Store/Google Play for updates
   - Install the latest version

2. Restart your device:
   - Sometimes a simple restart helps

3. Clear app cache (Android):
   - Settings > Apps > MicroBreaks > Storage
   - Clear Cache (not Clear Data)

4. Reinstall:
   - If nothing else works, try reinstalling
   - Note: This will reset your break history

Could you also share:
- Your device model (e.g., iPhone 14, Samsung S23)
- OS version (e.g., iOS 17.2, Android 14)
- When the crash happens (e.g., on launch, during exercise)
- Any error messages you see

This information helps us investigate and fix the issue.

Best regards,
[Agent Name]
MicroBreaks Support
```

## Response Guidelines

1. **Response Time:** Within 24-48 hours
2. **Tone:** Friendly, helpful, professional
3. **Personalization:** Use customer's name when provided
4. **Empathy:** Acknowledge frustration for issues
5. **Action:** Always provide next steps or solutions
6. **Follow-up:** Check back on unresolved issues
